Project Description

The Restaurant Ordering System is a Java-based console application developed using Object-Oriented Programming (OOP) principles. The system allows users to view menu items, place food and drink orders, and calculate the total bill through an interactive command-line interface.

This project demonstrates core Java concepts such as encapsulation, inheritance, polymorphism, abstraction, constructors, method overriding, exception handling, packages, and the use of ArrayList collections.

The application is organized into multiple classes including menu items, food items, drinks, orders, and restaurant services. User input is handled using the Scanner class, while try-catch blocks are used to manage invalid input safely.

Features
Display restaurant menu items
Add food and drink items to an order
Calculate total order price
Handle invalid user input
Organized package structure
Demonstrates Java OOP concepts
